# 🧠 Deep Learning Models Laboratory

Welcome to the **Deep Learning Models Laboratory** repository. This project contains a comprehensive set of assignments and data related to the study and implementation of various Deep Learning architectures.

## 📌 Project Overview
This repository serves as a practical guide to mastering Deep Learning techniques, starting from fundamental data manipulation to building complex generative models. Each assignment is implemented in a Jupyter Notebook, providing step-by-step code and visualizations.

## 🚀 Assignments Table

| # | 📂 Assignment Title | 📝 Description | 🛠️ Tech Stack |
|:---:|:---|:---|:---|
| 01 | [**Data Exploration**](./Assignment-01-Data-Exploration.ipynb) | Exploratory Data Analysis (EDA) and visualization using real-world datasets. | `Pandas`, `Matplotlib`, `Kagglehub` |
| 02 | [**TensorFlow Basics**](./Assignment-02-TensorFlow-Basics.ipynb) | Fundamental operations in TensorFlow including tensor manipulation and basic modeling. | `TensorFlow`, `NumPy` |
| 03 | [**NN from Scratch**](./Assignment-03-Neural-Network-from-Scratch.ipynb) | Manual implementation of a Neural Network (MLP) with backpropagation for XOR logic. | `NumPy`, `Matplotlib` |
| 04 | [**CNN Classification**](./Assignment-04-CNN-Image-Classification.ipynb) | Image classification using Convolutional Neural Networks (CNNs). | `Keras`, `TensorFlow` |
| 05 | [**RNN & LSTM**](./Assignment-05-RNN-LSTM.ipynb) | Sequence modeling and time-series analysis using RNN and LSTM layers. | `Keras`, `LSTMs` |
| 06 | [**Transfer Learning**](./Assignment-06-Transfer-Learning-ResNet50.ipynb) | Leveraging pre-trained ResNet50 models for efficient image classification. | `ResNet50`, `Fine-tuning` |
| 07 | [**GAN Generation**](./Assignment-07-GAN-Image-Generation.ipynb) | Generating synthetic data using Generative Adversarial Networks (GANs). | `GANs`, `Generative Models` |

- Jupyter Notebook / Lab
- TensorFlow / Keras
- NumPy, Pandas, Matplotlib, Seaborn
